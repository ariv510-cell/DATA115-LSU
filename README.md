# data115microbitsensors

![Data 115](card.png)

Maker Lab micro:bit extension for DATA 115.

## Features
- Raw LCD1602 support (no backpack)
- OLED graphics and animations
- Sensor helper functions
- MPU6050 IMU support
- Rotary encoder support
- Environmental sensors
- Motion sensors
- Lighting modules
- Datalogger integration
- Dashboard displays
- Custom Maker Lab blocks

## Supported Hardware
- BBC micro:bit
- LCD1602 Displays
- SSD1306 OLED Displays
- MPU6050 / GY-521
- Rotary Encoders
- DHT11 Sensors
- Ultrasonic Sensors
- LEDs and RGB modules
- Buzzers
- Shock / Tilt sensors
- Light sensors
- Sound sensors

## Installation
1. Open MakeCode for micro:bit
2. Go to Extensions
3. Search:
   data115microbitsensors
4. Add the extension
